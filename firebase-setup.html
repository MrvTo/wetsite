üìã Step-by-Step Setup

1. Create Firebase Project

  1. Go to Firebase Console
  2. Click "Create a project"
  3. Name it "wet-eyetracking" (or your choice)
  4. Enable Google Analytics (optional)
  5. Click "Create project"

2. Enable Authentication

  1. In Firebase Console, go to "Authentication"
  2. Click "Get started"
  3. Go to "Sign-in method" tab
  4. Enable "Email/Password"
  5. Save changes

3. Enable Firestore Database

  1. Go to "Firestore Database"
  2. Click "Create database"
  3. Start in "Test mode" (we'll secure it later)
  4. Choose your preferred location
  5. Click "Done"

4. Get Firebase Config

  1. Go to Project Settings (gear icon)
  2. Scroll down to "Your apps"
  3. Click "Web" ( icon)
  4. Register app name: "wet-frontend"
  5. Copy the configuration object

Your config will look like this:

const firebaseConfig = { apiKey: "AIzaSyC...", authDomain: "your-project.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123..." };

5. Update Your Frontend Files

Replace the Firebase config in js/firebase-config.js:

// Replace this in js/firebase-config.js const firebaseConfig = { // Paste your actual config here };

6. Update Registration Page

Update register.html to use Firebase:

<!-- Replace the api.js script with firebase-api.js --> <script type="module"> import './js/firebase-api.js'; </script>

7. Deploy to Netlify

  1. Push your code to GitHub
  2. Go to Netlify
  3. Click "New site from Git"
  4. Connect your GitHub repo
  5. Set build settings:
    • Build command: npm install
    • Publish directory: .
  6. Click "Deploy site"

8. Configure Environment Variables

  1. In Netlify, go to Site settings ‚Üí Environment variables
  2. Add these variables:
    • FIREBASE_PROJECT_ID = your-project-id
    • FIREBASE_API_KEY = your-api-key
    • FIREBASE_AUTH_DOMAIN = your-project.firebaseapp.com
